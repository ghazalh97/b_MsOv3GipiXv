# Quick Start Guide

## 🚀 Get Running in 2 Minutes

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

The app will open at: **http://localhost:5173**

---

## 📋 What to Do First

### 1. **Create Your First Goal**
- Click **"+ New Goal"** button
- Fill in:
  - Title: "Learn React"
  - Category: "Study"
  - Type: "Daily"
  - Target: "30" (minutes per day)
  - Start Date: Today
- Click **Save**

### 2. **Log Some Progress**
- Go to Dashboard
- Click the ✅ button on your goal
- Progress increases by 1
- Earn **20 XP** per log

### 3. **Explore the App**
- **Dashboard** - See overall stats
- **All Goals** - Filter and search
- **Categories** - View progress by category
- **Settings** - Change language (EN/FA)

---

## 🌐 Language Switching

Click the **"EN"** or **"فا"** button in the top right:
- The entire UI switches language
- Layout switches from LTR to RTL
- All text is translated

---

## 💡 Tips

### Progress Tracking
- Click ✅ to add 1 to progress
- Or go to goal details to log custom amounts
- When you reach target → **Goal auto-completes** ✓

### Streaks
- Log progress on consecutive days
- Streak increases by 1 each day
- Breaks if you miss a day

### XP System
- **+20 XP** = Log progress
- **+100 XP** = Complete goal
- View total XP on dashboard

### Pause Goals
- Click ⏸️ to pause tracking
- Click ▶️ to resume
- No XP earned while paused

## 🐛 Troubleshooting

**App won't start?**
```bash
rm -rf node_modules package-lock.json
npm install
npm run dev
```

**Data disappeared?**
- Check Settings → Reset Data
- Check if private browsing is on
- Browser may have cleared LocalStorage

**Language not changing?**
- Clear browser cache
- Check if JavaScript is enabled
